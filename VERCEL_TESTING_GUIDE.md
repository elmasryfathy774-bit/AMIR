# 🚀 كيفية اختبار التطبيق على Vercel

## الطريقة 1️⃣: الاختبار المباشر من المتصفح

### الخطوة 1: افتح التطبيق
```
1. اذهب إلى: https://amir-beta.vercel.app
2. ستظهر الصفحة الرئيسية (index.html)
```

### الخطوة 2: اختبر الملفات الجديدة

#### ✅ اختبار call.html (ملف المكالمات)
```
الرابط: https://amir-beta.vercel.app/call.html

المميزات التي يجب أن تشتغل:
1. زر الإضافة (➕) في الأسفل
2. نافذة اختيار الأصدقاء عند الضغط على الزر
3. أزرار كتم الصوت والسماعة
4. عرض حالة الاتصال
```

#### ✅ اختبار room.html (ملف الغرفة/الدردشة)
```
الرابط: https://amir-beta.vercel.app/room.html

المميزات التي يجب أن تشتغل:
1. Banner أزرق (إذا كان المستخدم في مكالمة) = "المستخدم في مكالمة أخرى"
2. خيار "إخفاء التقارير" في القائمة
3. صورة المستخدم واسمه في الأعلى
4. حقل كتابة الرسائل
```

#### ✅ اختبار chat.html (ملف الدردشات)
```
الرابط: https://amir-beta.vercel.app/chat.html

يجب أن:
1. يفتح قائمة المحادثات
2. لا يظهر أي خطأ
3. يعرض جهات الاتصال
```

## الطريقة 2️⃣: فتح Developer Tools (أدوات المطور)

### خطوات فتح Console للتحقق من الأخطاء:

**على Windows/Linux:**
```
F12 أو Ctrl + Shift + I
```

**على Mac:**
```
Cmd + Option + I
```

### التحقق من الأخطاء:
```
1. انقر على تبويب "Console"
2. ابحث عن أي رسائل خطأ حمراء
3. يجب أن ترى رسائل خضراء [v0] للأكواد الجديدة
```

---

## الطريقة 3️⃣: اختبار الميزات الجديدة

### ✅ اختبار Busy Status (حالة المشغول)
```
1. افتح room.html
2. إذا كان هناك اتصال نشط يجب أن تظهر:
   - Banner أزرق في الأعلى
   - نص: "المستخدم في مكالمة أخرى"
```

### ✅ اختبار Hide Read Receipts (إخفاء التقارير)
```
1. افتح room.html
2. انقر على القائمة (≡) في الأعلى
3. اختر "إخفاء التقارير"
4. الزر يجب أن يتغير (يصبح شفاف)
```

### ✅ اختبار Group Call Modal (نافذة المكالمة الجماعية)
```
1. افتح call.html
2. انقر على زر الإضافة (➕)
3. يجب أن تظهر نافذة تحتوي على:
   - قائمة الأصدقاء
   - checkboxes بجانب كل صديق
   - زر "بدء المكالمة"
4. اختر أصدقاء وانقر "بدء المكالمة"
```

### ✅ اختبار Image Compression (ضغط الصور)
```
1. افتح console (F12)
2. ابحث عن رسائل مثل:
   "Image compressed: 2500KB → 450KB"
```

## الطريقة 4️⃣: فحص في Browser DevTools

### تبويب Network:
```
1. F12 → Network tab
2. أعد تحميل الصفحة
3. يجب أن ترى الملفات تُحمل بنجاح:
   - call.html ✓
   - room.html ✓
   - chat.html ✓
```

### تبويب Elements:
```
1. F12 → Elements/Inspector
2. ابحث عن العناصر الجديدة:
   - class="modal-overlay-group" (نافذة المكالمة الجماعية)
   - id="busyBanner" (banner الحالة)
   - id="hideReadReceiptsBtn" (زر إخفاء التقارير)
```

## ✅ علامات النجاح

يجب أن ترى:
- ✅ لا توجد أخطاء حمراء في Console
- ✅ جميع الصور والملفات تُحمل (200 status)
- ✅ الأزرار الجديدة تظهر
- ✅ النوافذ الجديدة تفتح بدون مشاكل
- ✅ رسائل [v0] خضراء في Console

## ❌ إذا حصلت على أخطاء

### 404 Error:
```
السبب: الملف غير محمل على Vercel
الحل: انتظر 2-3 دقائق وأعد التحميل
```

### JavaScript Error:
```
السبب: خطأ في الكود
الحل: اطلب المساعدة وأرسل صورة Console
```

### Blank Page:
```
السبب: مشكلة في Firebase config أو Auth
الحل: تحقق من firebase-config.js
```
